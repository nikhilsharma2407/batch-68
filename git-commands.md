1. git init - initialise a repo
2.git add <filename> <filename2>...<filenameN>  -> git add index.html git-commands.md  - start tracking the resp file
3. git status - check the repo status
5. git add . - add all files to staging
6. git reset <filename> or . - unstage the changes
7. git checkout <filename> or . - undo the changes since last commit or staging
8. git commit -m <message> - commit the staged files
10. git checkout <branchname>  switch to branch
11. git checkout -b <branchname>  create a new branch
